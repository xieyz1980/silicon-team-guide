## 4.1 部署方式选择

### 🤔 有哪些部署方式？

OpenClaw提供**多种部署方式**，适合不同需求的用户：

| 部署方式 | 适合人群 | 难度 | 成本 | 特点 |
|---------|---------|------|------|------|
| **本地部署** | 个人用户、测试体验 | ⭐ 简单 | 免费 | 个人电脑运行，数据在本地 |
| **服务器部署** | 企业用户、团队协作 | ⭐⭐ 中等 | 服务器费用 | 24小时在线，团队共享 |
| **阿里云一键部署** | 国内用户、快速上线 | ⭐ 简单 | 按量付费 | 镜像市场一键安装 |
| **腾讯云轻量部署** | 国内用户、性价比高 | ⭐ 简单 | 按月付费 | 轻量服务器，开箱即用 |
| **火山引擎部署** | 字节生态用户 | ⭐ 简单 | 按量付费 | 字节云服务 |
| **SaaS平台** | 零运维用户 | ⭐ 极简 | 订阅制 | 免安装，网页即用 |

---

### 💡 如何选择？

**方案A：个人测试体验 → 本地部署**
- 在自己的Mac/Windows电脑上运行
- 适合：想先体验功能的个人用户
- 缺点：电脑关机后不可用

**方案B：个人长期使用 → 阿里云/腾讯云一键部署**
- 使用国内云服务器，24小时在线
- 适合：需要稳定服务的个人用户
- 成本：约30-100元/月

**方案C：企业团队使用 → 服务器部署或SaaS**
- 专业服务器部署或直接使用SaaS
- 适合：企业用户、团队协作
- 成本：根据规模

## 4.2 方式一：本地部署（个人电脑）

### 适用场景
- 想在个人电脑上体验OpenClaw
- 不需要24小时在线
- 数据存在本地

### 系统要求

**Mac用户**：
- macOS 10.15 或更高版本
- 4GB 内存
- 2GB 可用磁盘空间

**Windows用户**：
- Windows 10/11
- 4GB 内存
- 2GB 可用磁盘空间

**Linux用户**：
- Ubuntu 18.04+ / CentOS 7+
- 4GB 内存
- 2GB 可用磁盘空间

---

### 安装步骤

#### Mac/Linux 用户

**步骤1：打开终端**

Mac用户：打开"启动台" → 搜索"终端" → 打开

Linux用户：按 `Ctrl+Alt+T` 打开终端

**截图示例**（此处应有截图：Mac终端界面）

---

**步骤2：运行安装命令**

在终端中输入以下命令：

```bash
curl -fsSL https://openclaw.ai/install.sh | bash
```

**截图示例**（此处应有截图：终端中输入命令）

---

**步骤3：等待安装完成**

看到 "Installation complete!" 表示安装成功

**截图示例**（此处应有截图：安装成功提示）

## 4.3 方式二：阿里云一键部署（推荐）

### 适用场景
- 国内用户，需要稳定访问
- 希望24小时在线
- 不想自己配置服务器

### 前提条件
- 阿里云账号（需实名认证）
- 支付宝或银行卡（用于支付）

---

### 部署步骤

**步骤1：登录阿里云**

打开 https://www.aliyun.com 并登录

**截图示例**（此处应有截图：阿里云首页）

---

**步骤2：进入云服务器ECS**

点击"产品" → "云计算基础" → "云服务器ECS"

**截图示例**（此处应有截图：ECS入口）

---

**步骤3：创建实例**

点击"创建实例"，选择：
- **地域**：选择离你最近的（如华东1-杭州）
- **实例规格**：推荐"共享标准型s6 1核2G"（约30元/月）
- **镜像**：在"镜像市场"搜索"OpenClaw"，选择官方镜像
- **存储**：默认40GB即可

**截图示例**（此处应有截图：选择OpenClaw镜像）

---

**步骤4：配置网络**

- **网络类型**：专有网络（默认）
- **公网带宽**：按流量付费（省钱）或固定带宽（稳定）
- **安全组**：开放端口 18789（OpenClaw默认端口）

**截图示例**（此处应有截图：安全组配置）

---

**步骤5：设置登录方式**

选择"密码"或"密钥对"，设置 root 密码

⚠️ **重要**：密码要复杂，记得保存！

**截图示例**（此处应有截图：设置密码）

---

**步骤6：确认订单并创建**

查看配置，点击"确认订单"，支付后等待实例创建（约1-2分钟）

**截图示例**（此处应有截图：实例创建中）

---

**步骤7：获取公网IP**

实例创建完成后，在实例列表中看到"公网IP"地址

**截图示例**（此处应有截图：公网IP位置）

---

**步骤8：登录服务器**

Mac/Linux用户终端：
```bash
ssh root@你的公网IP
```

Windows用户使用PuTTY或Xshell：
- 主机名：你的公网IP
- 用户名：root
- 密码：刚才设置的密码

**截图示例**（此处应有截图：SSH登录成功）

---

**步骤9：验证OpenClaw安装**

登录后输入：

```bash
openclaw --version
```

看到版本号表示安装成功！

---

**步骤10：启动Gateway**

```bash
openclaw gateway start
```

✅ **阿里云部署完成！**

你的OpenClaw现在可以通过 `http://你的公网IP:18789` 访问

## 4.8 安装后必做：初始配置

无论哪种部署方式，安装完成后都需要进行初始配置：

### 步骤1：配置AI模型

OpenClaw需要连接AI模型才能工作，支持：
- OpenAI (GPT-4/GPT-3.5)
- Anthropic (Claude)
- DeepSeek (国产)
- GLM (智谱AI)
- OpenRouter (多模型聚合)

**配置方法**：

```bash
openclaw config set model.provider openrouter
```

## 4.9 常见问题排查

### ❌ 问题1：安装命令执行失败

**症状**：运行安装命令后报错

**解决**：
1. 检查网络连接
2. 尝试使用代理/VPN
3. 手动下载安装包：https://github.com/openclaw/openclaw/releases

---

### ❌ 问题2：Gateway启动失败

**症状**：运行 `openclaw gateway start` 后报错

**解决**：
1. 检查端口18789是否被占用：`lsof -i :18789`
2. 检查配置文件：`cat ~/.openclaw/openclaw.json`
3. 查看日志：`openclaw logs`

---

### ❌ 问题3：无法访问Web界面

**症状**：浏览器打不开 http://IP:18789

**解决**：
1. 检查Gateway是否运行：`openclaw gateway status`
2. 检查防火墙/安全组是否开放18789端口
3. 检查IP地址是否正确

---

### ❌ 问题4：阿里云/腾讯云镜像找不到OpenClaw

**解决**：
1. 在镜像市场搜索"OpenClaw"或"AI"
2. 如果找不到，选择纯净的Ubuntu镜像，然后手动安装
3. 手动安装命令见4.2节
